Relevant Courses

Below is a list of relevant computer science, computer engineering, and math courses I have taken so far or currently taking at Penn State.

Computer Science Courses

• CMPSC 465: Data Structures and Algorithms (In progress)

Fundamental concepts of computer science: data structures, analysis of algorithms, recursion, trees, sets, graphs, and sorting.

• CMPSC 311: Introduction to Systems Programming (In progress)

This course revolves around understanding computer systems, recognizing existing system software and software standards (UNIX toolset), comprehending "real code" (Linux OS kernel and GNU utilities/libraries), and understanding system performance. Topics include Unix system programming in C, organization of programs and data, program analysis and support tools, software standards, and common system functions.

• CMPSC 297: Hands-on Web Development (In progress)

Special topics course focusing on the design and implementation of web applications using programming languages, such as Python, Javascript, and SQL, and frameworks like Bootstrap, Flask, and Django. Covered topics include database design, scalability, security, UI/UX, and APIs.

• CMPSC 297: Blockchain - Deep Dive (In progress)

Special topics course centering around blockchain technology and Web3 development in the modern world. The course covers the basics of coding a blockchain and the usage of smart contracts to create decentralized web applications (Dapps). Languages, libraries, and frameworks utilized include Javascript, Express.js, React.js, and Solidity.

• CMPSC 221: Object Oriented Programming with Web-Based Applications

This course covers advanced object-oriented principles and their application to web-based, net-centric computing. Major topics include virtual machines, intermediate code generation, GUI design, event handling, and server-side programming with database queries. Languages used include Java and SQL.

• CMPSC 132: Programming and Computation II: Data Structures

Introductory course that builds upon the foundations of programming and computation by presenting data structures (linked lists, stacks, queues, trees, graphs), the foundations of object-oriented programming, the analysis of algorithms, and programming techniques, such as data abstraction, polymorphism, and higher-order functions. Taught in Python.

Computer Engineering Courses

• CMPEN 270: Digital Design: Theory and Practice

Introduction to digital systems and their design. This course teaches the theoretical foundation for digital systems and introduces combinational devices, logic gates, sequential devices, latches, and flip flops, along with design techniques, methods, and tools. Laboratory work includes the design, construction, and debugging of digital circuits.

Mathematics Courses

• STAT 414: Introduction to Probability Theory (In progress)

This course incorporates calculus into probability to describe the uncertainties present in real applications. Topics include probability spaces, discrete and continuous random variables, transformations, expectations, generating functions, conditional distributions, the law of large numbers, and central limit theorems.

• MATH 230: Calculus and Vector Analysis

Three-dimensional analytic geometry, vectors in space, partial differentiation, double and triple integrals, and integral vector calculus.

• MATH 220: Matrices

Systems of linear equations, matrix algebra, eigenvalues and eigenvectors, linear systems of differential equations.
